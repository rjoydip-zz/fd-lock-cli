# fd-lock-cli

[![Build Status](https://travis-ci.com/rjoydip/fd-lock-cli.svg?branch=master)](https://travis-ci.com/rjoydip/fd-lock-cli)

> Cli for cross-platform lock on a file using a file descriptor to it.

## Run

```shell
$ npm i -g fd-lock-cli
$ fd-lock
```
